Mini-course compositional data analysis 2024

Date: Jan 9-10 2024

Room: Location TBA

Instructors: Christian L. Müller, Stefanie Peschel, Viet Tran

Welcome!

  • The goal of the workshop is to help participants understand and apply concepts of compositional data analysis
  • Workshop material can be found in the workshop github repository (TBA).

Learning Objectives

At the end of the course, participants will be able to

  • understand key concepts of compositional data analysis
  • apply concepts of compositional data analysis in R

Pre-requisites

  • Basic familiarity with R (e.g., tutorial R for ecology)
